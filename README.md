# Book with Reminders

# Installation 

1. Clone Project
````sh
git clone https://github.com/imrantechwiz/Book.git
````
2. Composer install
````sh
composer install
````
3. Copy `.env.example` file and create `.env` file
4. Add Database Credential like yours in `.env`
```php
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_db_name
DB_USERNAME=root
DB_PASSWORD=
```
5. Go to the project - 
```sh
cd Book
```
6. Run Project inside that directory - 
````sh
php artisan serve
````
7. Open in Browser 
````sh
http://localhost:8000
````
